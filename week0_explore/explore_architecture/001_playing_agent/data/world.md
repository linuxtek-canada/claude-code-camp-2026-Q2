## The Bakery
- Location: North of Main Street (western segment)
- Navigation: Main St #1 → west → Market Square → west → Main Street (west) → north
- Exits: south
- NPCs: cityguard, the baker
- Features: bread and Danish on shelves, small sign on counter
- Menu:
  - A danish pastry - 7 gold (Unlimited)
  - A bread - 14 gold (Unlimited)
  - A waybread - 72 gold (Unlimited)
- Commands: `list` to see menu, `buy <item>` to purchase

## Movement Commands
- NORTH / SOUTH / EAST / WEST / UP / DOWN
- n / s / e / w / u / d (short form)

## Key Commands
- look / l - examine current room
- exits - list exits
- i / inventory - check inventory
- score - check stats
- help - show command list
- hindex <word> - search help index
- examine <object> - examine an object
- list - shop listing
- buy <item> - purchase from shop
- ask <npc> - ask NPC a question
